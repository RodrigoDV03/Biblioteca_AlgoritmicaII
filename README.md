Recursos Usados:
    - Python 3.10.7
    - DB Browser

Librerías usadas:
    - SQLite3
        Instalar Window: Escribir en terminal "pip Install pysqlite3"
        Instalar Linux: Escribir en terminal: "sudo apt install sqlite3"
    - ABC
    - werkzeug
        Instalar Windows: pip install werkzeug
